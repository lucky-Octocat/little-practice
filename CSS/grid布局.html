Simple grid example

  • 默认情况下 是一列，需要通过 grid-template-columns 来指定 列 数
    1. 需要几列 就指定几列的宽度
    2. 如果需要很多列，可以使用 repeat(12, 50px); 十二列, 每列 宽50px
    3. 如果需要 平分 父元素的宽度 使用 fr， fr 是 fraction，分数的意思
    4. 还可以 常数与fr 混合使用
  • 行：不指定 的情况下 列排不开就会自动换行
    1. grid-template-rows 属性 来指定 行高
    2. grid-auto-rows 未指定高度的行 会使用这个属性【优先级低一级】
  • 缝隙：
    1. 行 的间隙 grid-column-gap: 10px;
    2. 列 的间隙 grid-row-gap: 5px;
    3. 还可以简写： gap
